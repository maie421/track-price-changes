# track-price-changes
http://pricetracker.site/ </br>
쿠팡 밀키트 가격 변동 추적 사이트

## 프로젝트 목적
쿠팡 파트너스 활동을 위해 해당 사이트 개발을 진행하였습니다.<br>
쿠팡 로켓프레시에 있는 상품 가격 변화를 알 수 있도록 하여 사용자를 유도합니다.

## 프로젝트 기간
2023.12.02 ~ 

## 1차 프로토타입 배포 
2023.12.02 ~ 2023.12.13
로켓프레시에 있는 상품들이 많아 크롤링하여 전부다 데이터를 저장을 할려고 하였지만
1. 해당 데이터들을 크롤링 하는데 오래 걸리지 않을까?
2. 처음 부터 많은 데이터들을 서버가 버틸 수 있을까?

에 대한 의문점이 생겨 점점 시장이 커지고 있는 밀키트를 데이터 수집하기로 결정하였습니다.<br>
